# 🚀 LoanAI - Smart Loan Repayment
> *Intelligent, AI-powered loan repayment optimization platform.*

**LoanAI** is a comprehensive, modern web application designed to help users analyze their loans and create personalized repayment strategies. By leveraging Artificial Intelligence, LoanAI empowers users to save time and money through optimized financial planning.

## ✨ Key Features

- 🤖 **AI-Powered Analytics**: Automatically analyzes your loans to create a personalized repayment strategy that saves you money on interest.
- 🧮 **Quick Loan Calculator**: Get instant estimates of your monthly payments and see how different repayment strategies impact your timeline in real-time.
- 📅 **Smart Payment Scheduling**: Automatically schedule payments, receive reminders, and strategically plan extra payments to reduce your principal faster.
- 💬 **24/7 AI Assistant**: Chat with an integrated AI assistant to get instant answers to your loan questions, personalized advice, and step-by-step financial guidance.
- 📊 **Interactive Dashboard**: Visualize your loan data, track your savings projections, and monitor your journey to financial freedom with beautiful charts.
- 🛡️ **Secure & Free**: No credit check required, completely free analysis, and you can cancel any time.

## 🛠️ Tech Stack

This project is built with modern web technologies:

- **Framework**: [Next.js 16](https://nextjs.org/) (React 19)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) (Shadcn UI style)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Integration**: [Google Generative AI (Gemini)](https://ai.google.dev/)
- **Charts & Visualization**: [Recharts](https://recharts.org/)
- **Forms & Validation**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed along with a package manager like `npm` or `pnpm`.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sahil-Kumar-Chaudhary/loan-repayment.git
   cd loan-repayment
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or if using pnpm
   pnpm install
   ```

3. **Environment Setup:**
   Create a `.env.local` file in the root directory and add your required environment variables (especially for the AI integration):
   ```env
   # Example for Google Gemini API
   GOOGLE_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

5. **View the App:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the application in action.
